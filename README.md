# portfolio-site

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Git Command for Subtree Push
git subtree push --prefix dist origin gh-pages


### Instructions

Make sure sitemap, LB.png, 404, CNAME, are all in gh-pages
